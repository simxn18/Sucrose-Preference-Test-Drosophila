# Bonsai Workflow
This folder contains a Bonsai Workflow for tracking the X coords of a single fly for posterior analysis.

<img width="869" height="181" alt="SPT" src="https://github.com/user-attachments/assets/8dd71211-1108-472f-915d-5f551ee2f22c" />


## Process steps:
1. Video reading and transforming to grayscale.
2. Cropping ROIs for individual analysis.
3. Binary region analysis and tracking
4. Saving data to CSV.

The output of this workflow serves as the input for the SPT analysis script. An example is in /data folder.
